# Website_User_Insights

Brief SQL project on cleaning and gaining insight into user data

### Overview
- Goal: Fix nulls/dupes/types; surface core trends.

### Methods
- Cleaning: trims, type coercion, category normalization, dedupe via window functions.

### Results (highlights)
- Duplicates reduced
- Categories consolidated
- Averages and conversion rates calculated
- User Journey calculated

### How to Run
1) Load CSV into SQL program of your choice
2) Open a query
3) Copy and paste various queries and explore
